CouchDB Bind_Address & Port

To change the bind address

data = {:section => "httpd",
              :key => "bind_address",
                :value => "192.168.1.118"}
 
Couchdb.set_config(data, auth_session)

This only changes CouchDB’s bind_address, it doesn’t change the address used by Leanback to make http requests. Leanback uses ’127.0.0.1′ by default, to change Leanback’s address:

Couchdb.address = "192.168.1.118"

This means that Leanback will use the address “192.168.118″ to make all future requests.

The complete code for changing the bind address will be:

data = {:section => "httpd",
              :key => "bind_address",
                :value => "192.168.1.118"}
 
Couchdb.set_config(data, auth_session)
Couchdb.address = "192.168.1.118"

To change the port#

data = {:section => "httpd",
              :key => "port",
                :value => "6980" }
Couchdb.set_config(data,auth_session) 
Couchdb.port = "6980"

The last line

Couchdb.port = "6980"

Will make Leanback use the port# 6980 for all future requests.
