Count documents by key

First we login to CouchDB and generate the auth_session token for making requests.

hash = Couchdb.login(username = 'samson',password ='sam123') 
auth_session =  hash["AuthSession"]

To count the number of documents with a key (example: to count the number of documents with email = nancy@mail.com )

Couchdb.count({:database => 'contacts', :email => 'nancy@mail.com'} , auth_session)  
 
# => 1

This returns an integer of the number of documents that match the key.

To get the number of contacts with lastname = ‘Smith’

Couchdb.count({:database => 'contacts', :lastname => 'Smith'},auth_session)  
 
# => 2

To count all female contacts

Couchdb.count({:database => 'contacts', :gender => 'female'},auth_session) 
 
# => 8

If you are running an admin party (i.e you have no admins in CouchDB), simply use the same methods, with no auth_session parameter.

Couchdb.count( :database => 'contacts', :gender => 'female')

How it works
The method

Couchdb.count({:database => 'contacts', :email => 'nancy@mail.com'}, auth_session)

sends a

GET http://127.0.0.1:5984/contacts/_design/email_counter/_view/count_email?key="nancy@mail.com"

This returns the number of contacts with email = ‘nancy@mail.com’. If the design_document (_design/email_counter) and the view (count_email) doesn’t exist, Leanback will create it in the database. See the source for the generated design_document;

{
   "_id": "_design/email_counter",
   "_rev": "1-d5ec21ea6ce87f99a511dd02c196ce8f",
   "language": "javascript",
   "views": {
       "count_email": {
           "map": "function(doc){ if(doc.email) emit(doc.email,null);}",
           "reduce": "_count"
       }
   }
}

If you ran

Couchdb.count({:database => 'contacts', :lastname => 'Smith'},auth_session)  
Couchdb.count({:database => 'contacts', :gender => 'female'},auth_session)

you will notice the following design_documents have been added to the database

_design/lastname_counter
_design/gender_counter
